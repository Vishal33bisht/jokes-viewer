# 😂 Jokes Viewer

A simple and fun web application to view random jokes using an API. Built with HTML, CSS, and JavaScript.

---

## 🚀 Features

- Fetch random jokes instantly
- Clean and responsive UI
- Lightweight and fast
- Beginner-friendly project structure

## ⚡ Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Vishal33bisht/jokes-viewer.git
```

### 2️⃣ Navigate to the project folder

```bash
cd jokes-viewer
```

### 3️⃣ Open in browser

Simply open `index.html` in your browser.
